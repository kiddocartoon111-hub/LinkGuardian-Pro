How to get Telegram alerts?

After signing up, message @BotFather to create a bot, get token, and add to Railway variables. Then get your chat ID by messaging the bot and visiting api.telegram.org/bot/getUpdates.

What does multi-layer mean?

We try direct request first, then ScraperAPI if blocked. No browser fallback initially, but planned.

Out-of-stock false positives?

We use keywords; some sites may trigger incorrectly. Contact us to improve.
